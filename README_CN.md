# radp-bash-framework

## 安装

### Homebrew

```bash
brew tap xooooooooox/radp
brew install radp-bash-framework
```

安装后，可以通过以下方式获取 `run.sh` 的路径：

```bash
source "$(radp-bf --print-run)"
```
